# توحيد روتين صفحات الأحوال المدنية مع صفحات الجوازات

## نظرة عامة
تم توحيد جميع صفحات مجلد `civil_status` لتتبع نفس الروتين والبنية المستخدمة في صفحات مجلد `passports`.

## التحديثات المنفذة

### 1. إضافة سكريبت main.js
تم إضافة السكريبت `../js/main.js` في نهاية جميع الصفحات قبل إغلاق وسم `</body>`:

```html
<!-- Bootstrap Bundle JS -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
<!-- Custom JS -->
<script src="../js/main.js"></script>
</body>
```

### 2. الصفحات المحدثة
تم تحديث الصفحات التالية:

✅ **index.html** - الصفحة الرئيسية
✅ **id_requests.html** - طلبات الهوية الوطنية
✅ **life_events.html** - تسجيل الواقعات (ميلاد/وفاة)
✅ **family_registry.html** - سجل الأسرة
✅ **verifications.html** - التحقق من البيانات
✅ **review_requests.html** - مراجعة الطلبات والشكاوى

## البنية الموحدة

### هيكل الصفحة القياسي:

```html
<!DOCTYPE html>
<html lang="ar" dir="rtl">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>[عنوان الصفحة]</title>
    
    <!-- Bootstrap 5 CSS (RTL) -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.rtl.min.css">
    
    <!-- Font Awesome -->
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
    
    <!-- Custom CSS -->
    <link rel="stylesheet" href="../css/style.css">
    
    <style>
        :root {
            --primary-color: #10b981; /* للأحوال المدنية */
            --primary-hover: #059669;
        }
        /* أنماط مخصصة للصفحة */
    </style>
</head>

<body>
    <!-- Sidebar -->
    <div class="sidebar">
        <!-- محتوى القائمة الجانبية -->
    </div>

    <!-- Main Content -->
    <div class="main-content">
        <!-- Top Navbar -->
        <div class="top-navbar">
            <!-- شريط التنقل العلوي -->
        </div>
        
        <!-- محتوى الصفحة -->
    </div>

    <!-- Bootstrap Bundle JS -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
    
    <!-- Custom JS -->
    <script src="../js/main.js"></script>
    
    <!-- سكريبتات مخصصة للصفحة (اختياري) -->
</body>

</html>
```
